# 🚀 Bump

Bump یک قالب وب‌سایت مدرن و واکنش‌گرا (Responsive Landing Page) است که با استفاده از HTML5، CSS3 و Bootstrap توسعه داده شده است. این پروژه با هدف نمایش خدمات، دوره‌های آموزشی، پلن‌های قیمت‌گذاری و ارتباط با کاربران طراحی شده و می‌تواند به عنوان پایه‌ای مناسب برای وب‌سایت‌های آموزشی، استارتاپی و شرکتی مورد استفاده قرار گیرد.

## ✨ ویژگی‌های پروژه

* طراحی کاملاً واکنش‌گرا (Responsive)
* رابط کاربری مدرن و جذاب
* استفاده از Bootstrap 5
* منوی ناوبری حرفه‌ای
* دکمه بازگشت به بالای صفحه (Scroll To Top)
* کارت‌های معرفی خدمات با افکت‌های Hover
* انیمیشن اختصاصی برای کارت‌ها
* بخش معرفی امکانات و مزایا
* جدول قیمت‌گذاری (Pricing Plans)
* اسلایدر تصاویر (Carousel)
* فرم تماس با ما
* فوتر کامل با لینک‌های کاربردی
* استفاده از Bootstrap Icons
* طراحی RTL مناسب زبان فارسی

## 🛠️ تکنولوژی‌های استفاده شده

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons

## 📂 بخش‌های موجود در پروژه

### Header & Navigation

* منوی ناوبری واکنش‌گرا
* منوی کشویی محصولات
* هدر ثابت (Sticky Header)

### Hero Section

* معرفی اصلی وب‌سایت
* دکمه‌های فراخوانی به اقدام (CTA)

### Features Section

* نمایش امکانات و قابلیت‌ها
* کارت‌های معرفی خدمات

### Pricing Section

* پلن مقدماتی
* پلن متوسط
* پلن پیشرفته

### Testimonial / Showcase Section

* اسلایدر تصاویر و محتوا
* نمایش نمونه‌ها و امکانات

### Contact Section

* فرم ارسال پیام
* دریافت ایمیل کاربران

### Footer

* لینک‌های دسترسی سریع
* معرفی دوره‌ها
* شبکه‌های اجتماعی

## 🎨 ویژگی‌های طراحی

* افکت Hover پیشرفته روی کارت‌ها
* انیمیشن چرخشی حاشیه کارت‌ها
* استفاده از SVG Wave Divider بین سکشن‌ها
* طراحی مدرن با رنگ‌بندی Bootstrap Primary
* تجربه کاربری روان و زیبا

## 🚀 نحوه اجرا

```bash
git clone https://github.com/Aref18/bump.git
```

سپس فایل `index.html` را در مرورگر اجرا کنید.
